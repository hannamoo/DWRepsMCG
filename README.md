# Algorithmic method to find primitive central idempotents of S_3

Let $A(S_3)$ be annular algebra with $S_3$ labeled string nets. A basis is given by $V_{g, h}$. Then, $V_{a, b} \cdot V_{c,d} = \delta_{bab^{-1}, c} V_{a, db}$.

1. Enumerate $\{ V_{g,h} \} as {V_j}_{j=1}^{\mid G\mid ^2}$

2. Fill matrix $Z$ such that $Z_{(i-1)36 + k, j} = d_{ij}^k - d_{ji}^k$

3. Find the basis of $ker(Z)$
